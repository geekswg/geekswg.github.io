# VSCode插件推荐

Visual Studio Code的强大之处在于其丰富的插件生态系统。本文推荐一些实用的VSCode插件。

## 前端开发插件

### 1. Auto Rename Tag
自动重命名配对的HTML/XML标签。

### 2. Bracket Pair Colorizer
为匹配的括号添加颜色，提高代码可读性。

### 3. ESLint
JavaScript代码质量检查工具。

### 4. Prettier
代码格式化工具，支持多种语言。

## 通用开发插件

### 1. GitLens
增强Git功能，显示代码作者和提交历史。

### 2. Live Server
启动本地开发服务器，支持热重载。

### 3. Thunder Client
轻量级REST API客户端。

### 4. Todo Tree
在代码中高亮显示TODO、FIXME等注释。

## 主题和外观

### 1. Material Icon Theme
Material Design风格的图标主题。

### 2. One Dark Pro
流行的暗色主题。

### 3. Peacock
为不同的工作区设置不同的颜色主题。

## 实用工具

### 1. Code Spell Checker
代码拼写检查器。

### 2. Settings Sync
同步VSCode设置到GitHub Gist。

### 3. Remote - SSH
通过SSH连接远程服务器进行开发。

这些插件可以显著提高开发效率和代码质量。
